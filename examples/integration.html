Sitio del cliente (demo)

Este HTML muestra cómo queda el widget CPIQ embebido en cualquier página. El botón flotante aparece abajo a la derecha.

Snippet de integración

Pegá este script justo antes de </body>:

<script
  src="https://widget.cpiq.com/loader.js"
  data-position="bottom-right"
  data-primary-color="#0B5FFF"
  data-panel-origin="https://panel.cpiq.com"
  data-client-id="demo"
  data-version="1"
  data-teaser-text="Chat con el Consejo"
  data-teaser-delay="1500"
  async></script>

API programática

Una vez cargado, el widget expone window.CPIQWidget:

Escuchar eventos (analytics)

window.addEventListener('cpiq:navigate', (e) => {
  console.log('click en opción', e.detail);
  // enviar a GA / Segment / Plausible
});

window.addEventListener('cpiq:toggle', (e) => {
  console.log('widget abierto?', e.detail.open);
});
